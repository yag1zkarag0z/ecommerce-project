🛒 React E-Commerce App

A full-stack e-commerce application built with React, featuring routing, backend integration, data fetching, data mutation, and automated testing.

🚀 Tech Stack


React 19 – UI library
React Router – Client-side routing
Vite – Build tool & dev server
CSS / React Hooks – Styling and state management
Backend API – Data fetching & mutation
Vitest – Automated testing
